Процесс простой, два шага каждый раз:
1 — Вносишь изменения в код у себя на компьютере, проверяешь локально через npm run dev
2 — Деплоишь на сайт:
npm run build
npm run deploy
Через 1–2 минуты изменения появляются на sortament.pro. Весь процесс занимает 2–3 минуты.
Для отладки — npm run dev запускает сайт локально на http://localhost:3000, там всё работает в реальном времени: сохранил файл → браузер обновился автоматически. На живой сайт пушишь только когда всё готово.


Всё отлично — сайт уже живёт по адресу http://sortament.pro/ и DNS проверка идёт.
Подожди 10–30 минут, потом страница обновится, DNS Check пройдёт и появится галочка Enforce HTTPS — включи её.
